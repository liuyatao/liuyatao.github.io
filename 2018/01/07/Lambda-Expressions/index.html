Java 8 Lambda Expressions

2018-01-07

匿名内部类的一个问题是：当一个匿名内部类的实现非常简单，比如说接口只包含一个函数，那么匿名内部类的语法有点笨拙且不清晰。在这种情况下，我们经常会是传递一个函数作为参数给另一个函数，比如当点击一个按钮的响应事件。Lambda表达式可以通过把函数当做函数的参数，代码（函数）当做数据（参数引用）的特性能够实现这一点。

Lambda表达式可以让只有一个方法的类更灵活。
